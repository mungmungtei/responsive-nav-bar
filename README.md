# responsive-nav-bar
## 반응형 이란?

**반응형 웹 디자인**(responsive web design, RWD)이란 하나의 웹사이트에서 PC, 스마트폰, 태블릿 PC 등 접속하는 디스플레이의 종류에 따라 화면의 크기가 자동으로 변하도록 만든 웹페이지 접근 기법을 말한다. ➡️ Content is like Water 🌊
